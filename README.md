一、方案标题
==
宠物喂食器
二、方案应用场景
==
地点：无人在家宠物喂食；<br>
功能：铲屎官出门在外家里的宠物无人照顾，或者铲屎官比较懒，这时可以通过手机在线操控机器进行自动投喂。
     当宠物粮不够，发出报警，提醒铲屎官投粮。
